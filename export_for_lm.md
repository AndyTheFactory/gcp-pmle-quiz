# Using Your Wrongly Answered Questions File with Notebook LM

If you have a file containing questions you answered incorrectly, you can leverage it to improve your learning with Notebook LM. Here’s how:

1. **Upload as a Source:**
    Add your file of wrongly answered questions to Notebook LM as a Source. This allows the tool to reference your specific knowledge gaps.

2. **Combine with Other Sources:**
    You can use this file alongside other study materials or sources. Notebook LM will integrate information from all your uploaded sources.

**Suggested starting sources**

Use these Google Cloud resources as a starting point; you can add other files or URLs (documentation, notebooks, blog posts, or personal notes) as Sources in Notebook LM.

```
https://docs.cloud.google.com/vertex-ai/docs/training/neural-architecture-search/suggested-workflow
https://docs.cloud.google.com/dataflow/docs/optimize-costs
```

You can add additional files or URLs to tailor sources to your study needs.

3. **Create Flashcards and Quizzes:**
    Use Notebook LM’s features to generate flashcards and quizzes based on your uploaded questions. This targeted practice helps reinforce concepts you previously struggled with.

4. **Deepen Your Understanding:**
    By focusing on areas where you made mistakes, you can efficiently address your weaknesses and solidify your knowledge.
